## Objective
Implement Convex database mutations and actions to simulate the multi-agent state machine. This will sequentially transition the run step from Observe to Evaluate, appending mock logs and seeding mock datasets so that all downstream dashboards and terminal components have data to display.

    - Step 5: Wait 1.5s, call `simulateStep` to update to "Evaluate" (Evaluation Agent), append logs, status: "completed", and seed final metrics output comparison.
  </action>
